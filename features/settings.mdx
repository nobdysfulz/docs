The Settings page (`/Settings`) is the central configuration hub for all user, billing, integration, and admin settings. It uses role-based access control to show different sections to regular users and administrators.

## Navigation

Settings uses a left sidebar navigation (`SettingsSidebar`) organized into sections, with tab selection via URL search parameters (`?tab=profile`).

## User Settings

### Account Settings

#### Profile
User account information and personal details:
- Name, email, phone
- Profile photo
- Brokerage information
- Bio and description

#### Market
Geographic market configuration:
- City, state, zip codes
- Market type and price ranges
- Territory boundaries
- Feeds into MORO intelligence scoring

#### Agent Intelligence
AI agent configuration and behavior settings:
- Agent response style preferences
- Knowledge base management
- Guidelines customization per agent

#### Mercury AI
Mercury AI integration settings for the activity feed and AI assistant.

#### Billing
Subscription and payment management with Stripe integration:
- **Current plan** - Plan name, billing interval (monthly/yearly), status (active, trialing, past_due, canceled)
- **Payment history** - Last/next payment amounts and dates, invoice links
- **Payment method** - Connected card display (brand, last 4 digits, expiration)
- **Platform credits** - Available balance, minutes used vs allocated, usage progress bar
- **Transaction history** - Recent credit transactions (refill, purchase, bonus, grant, reset)
- **Manage Billing Portal** - Links to partner Stripe Connect portal or platform billing portal

#### Credits & Usage
Dedicated credit management interface:
- Credit balance overview
- Usage history and tracking
- Credit purchase options

#### Referrals
Referral program management for inviting other agents to the platform.

#### Notifications
Notification preferences:
- Email notification settings
- Push notification settings
- Alert frequency configuration

#### Preferences
General user preferences:
- UI/UX settings
- Default views
- Content preferences

#### Security
Authentication and security settings:
- Password management
- Two-factor authentication
- Session management
- Login history

### Connections

#### Integrations
Third-party service connections:
- CRM integrations (Lofty, Follow Up Boss)
- Social media accounts
- Email service providers
- Calendar integrations

#### CRM
CRM-specific integration configuration:
- CRM sync settings (enable/disable `syncGoals`)
- Field mapping
- Sync frequency
- Link to main Integrations tab

#### Messaging
Messaging service setup:
- SMS provider configuration
- Email provider settings
- Messaging templates

### Onboarding

#### Setup Progress
Guided onboarding tracking:
- Completion status for each onboarding step
- Resume incomplete onboarding
- Re-run specific setup wizards

## Admin Settings

<Note>
Admin sections are only visible to users with admin roles.
</Note>

### Super Admin Panel
- **Admin Permissions** - Manage super admin access and roles

### Content Management
- **Content Topics** - Configure available content generation topics
- **Content Packs** - Manage pre-built content packs
- **Featured Packs** - Curate featured content packs for the Content Studio
- **AI Prompts** - Edit system and user prompt templates (`AiPromptConfig` entity)

### Call Center
- **Campaign Templates** - Configure outbound calling campaign templates
- **Agent Voices** - Manage ElevenLabs voice configurations for AI calls

### Training
- **Role-Play Scenarios** - Create and edit training scenarios
- **Client Personas** - Define AI opponent personalities
- **Objection Scripts** - Manage the objection script library

### System Management
- **Task Templates** - Configure recurring task templates by day of week
- **User-Partner Links** - Manage multi-tenant user-partner associations
- **Legal Documents** - Manage disclosures and legal text
- **Email Campaigns** - System-level email campaign management
- **Global Library** - Shared content library across all users
- **Push Notifications** - Send push notifications to users
- **System Monitoring** - Platform health and performance metrics
- **Error Logs** - Application error tracking and debugging
- **Feature Flags** - Toggle features on/off for rollouts
- **Integration Health** - Monitor status of all third-party integrations
- **Autopilot Monitor** - Monitor automated workflow execution
- **Data Export** - Export platform data for analysis

### Partner Admin
- **Partner Settings** - Configure partner branding, colors, display name, favicon, and meta description

## Technical Details

- All settings tab components are **lazy-loaded** using `React.lazy()` with `Suspense` boundaries
- Skeleton loading states are shown while tabs load
- Mobile devices render a separate settings layout via `MobileSettingsPage`
- Separate `MobileBillingPage` for billing on mobile
- Settings are persisted to their respective entities via the Base44 SDK
